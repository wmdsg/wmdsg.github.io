Java中正则表达式相关API介绍

利用正则表达式通常会有以下形式：

String RegExp="";
Pattern pattern=Pattern.compile(RegExp);
Matcher matcher=pattern.matcher(content);
while(matcher.find()){
    sout.matcher.group(0);
}

matcher.find()完成的任务：

  1. 根据指定的规则，定位满足的子字符串如（1998）
  2. 找到后，将子字符串开始的索引记录到matcher对象的group数组中
    1. group[0]=0,把该子字符串的结束索引+1，记录到group[1]=4
  3. 同时记录oldLast值记为子字符串的解释+1即为4，下次find时，就从oldLast开始匹配

再次找到时将group[0]和group[1]刷新，记录新的匹配串。

当RegExp有分组时，即有括号时：

例如(\\d\\d)(\\d\\d)

  1. 根据指定的规则，定位满足的子字符串如（1998）

  2. 找到后，将子字符串开始的索引记录到matcher对象的group数组中

    1. 总的子字符串group[0]=0,group[1]=4，
    2. 第一组字符串：group[2]=0，group[3]=2，第二组字符串：group[4]=2，group[5]=4
    3. 更多的分组继续往后记

示例：

public void RegExp(){
        String content="韩顺平教育，韩顺平学习，韩顺平同学";
        Pattern compile = Pattern.compile("韩顺平(?:教育|学习|同学)");
        Matcher matcher = compile.matcher(content);
        while(matcher.find()){
            System.out.println(matcher.group(0));
        }
    }
//返回结果：
韩顺平教育
韩顺平学习
韩顺平同学
