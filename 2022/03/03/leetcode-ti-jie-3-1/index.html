344. 反转字符串

较简单，双指针

class Solution {
    public void reverseString(char[] s) {
        int i=0;
        int j=s.length-1;
        while(i<j){
            char c=s[j];
            s[j]=s[i];
            s[i]=c;
            ++i;
            --j;
        }
    }
}

541. 反转字符串 II

分组之后分别反转即可。

class Solution {
    public String reverseStr(String s, int k) {
        char[] tmp=s.toCharArray();
        int len=s.length();
        int loop=len/(2*k);
        int mod=len%(2*k);
        int des=loop*2*k;
        for(int i=0;i<des;i+=(2*k)){
            int left=i;
            int right=left+k-1;
            reverse(left,right,tmp);
        }
        if(mod>0&&mod<k){
            int left=des;
            int right=len-1;
            reverse(left,right,tmp);
        }else if(mod>=k&&mod<(2*k)) {
            int left=des;
            int right=left+k-1;
            reverse(left,right,tmp);
        }
        return String.valueOf(tmp);
    }

    public void reverse(int left,int right,char[] tmp){
        while(left<right){
            char ch=tmp[right];
            tmp[right]=tmp[left];
            tmp[left]=ch;
            ++left;
            --right;
        }
    }
}

剑指 Offer 05. 替换空格

==StringBuilder遍历添加==

题目虽然是替换空格，不如我们换个思路，不在源字符串上下功夫，而是新开空间，遇到空格时再将空格换成指定字符串，如果不是空格，则原封不动移到答案中。

class Solution {
    public String replaceSpace(String s) {
        StringBuilder builder=new StringBuilder();
        for(int i=0;i<s.length();i++){
            if(s.charAt(i)==' '){
                builder.append("%20");
            }else {
                builder.append(s.charAt(i));
            }
        }
        return builder.toString();
    }
}

==双指针==

上一个方法时新开空间，注意到替换字符长度是空格的三倍，我们可以在源字符串上增加空间，先统计源字符串有几个空格，再增加相应位置。转成数组后，右指针指向新数组的长度，左指针指向原来的长度，一起向前遍历，若左指针碰到字符，则复制到右指针，若碰到空格，则右指针添加指定字符。

class Solution {
    public String replaceSpace(String s) {
        //双指针做法
        int len=s.length();
        for(int i=0;i<len;i++){//可通过+=这种方式来扩充空间。
            if(s.charAt(i)==' '){
                s+="  ";
            }
        }
        char[] strs=s.toCharArray();
        int left=len-1;
        int right=s.length()-1;
        while(left>=0){
            if(strs[left]==' '){
                strs[right--]='0';
                strs[right--]='2';
                strs[right--]='%';
            }else {
                strs[right--]=strs[left];
            }
            --left;
        }
        return String.valueOf(strs);
    }
}

151. 翻转字符串里的单词

这题用传统的双指针做法的话将会很难控制左右指针的移动。因为前后的单词字母个数不同，不如换个思路，以一种更全局的策略去审视这道题，这道题不就是要把字母反转吗，只不过反转后的字母要重新构成单词，于是我们可以先反转整个字符串，然后在根据空格为分隔符，挨个反转每个单词，使之构成一个正确的单词。

注意几个小细节

  • 输入的字符串s可以在前面，后面或单词间包含多于的空格，故需先做空格去除处理。
class Solution {
   public String reverseWords(String s) {
        //简单做法可以使用spilt分割后倒序相加
        //进阶做法是在源字符串上下功夫
        s=removeSpace(s);
        int start=0;
        int end=0;
        int len=s.length();
        char[] tmp=s.toCharArray();
        reverse(0,len-1,tmp);
        while(start<len){
            while(end<len&&tmp[end]!=' ') end++;
            reverse(start,end-1,tmp);
            start=end+1;
            end=start;
        }
        return String.valueOf(tmp);
    }

    //1.移除空格
    public String removeSpace(String original){
        StringBuilder sb=new StringBuilder();
        int start=0;
        int end=original.length()-1;
        while(start<=end&&original.charAt(start)==' ') start++;
        while (end>=start&&original.charAt(end)==' ') end--;
        for(int i=start;i<=end;i++){
            if(original.charAt(i)!=' '){
                sb.append(original.charAt(i));
            }else {
                sb.append(' ');
                while(original.charAt(i)==' ') i++;
                i--;
            }
        }
        return sb.toString();
    }

    //2.反转字符串
    public void reverse(int left,int right,char[] tmp){
        while(left<right){
            char ch=tmp[right];
            tmp[right]=tmp[left];
            tmp[left]=ch;
            ++left;
            --right;
        }
    }
}

剑指 Offer 58 - II. 左旋转字符串

class Solution {
    public String reverseLeftWords(String s, int n) {
        StringBuilder sb=new StringBuilder();
        char[] tmp=new char[n];
        for(int i=0;i<s.length();i++){
            if(i<n){
                tmp[i]=s.charAt(i);
            }else {
                sb.append(s.charAt(i));
            }
        }
        sb.append(tmp);
        return sb.toString();
    }
}

28. 实现 strStr()

==暴力匹配==

字符串匹配算法是比较重要的算法，最简单的就是暴力匹配，此题为了减少不必要的匹配，在一旦出现不匹配的字符时便停止循环，向后遍历。匹配下一个字符。

class Solution {
    public int strStr(String haystack, String needle) {
        int n = haystack.length(), m = needle.length();
        for (int i = 0; i + m <= n; i++) {
            boolean flag = true;
            for (int j = 0; j < m; j++) {
                if (haystack.charAt(i + j) != needle.charAt(j)) {
                    flag = false;
                    break;
                }
            }
            if (flag) {
                return i;
            }
        }
        return -1;
    }
}

==KMP算法==

KMP算法是解决字符串匹配的经典算法，其核心就在于理解next数组的含义以及如何求出next数组，关于KMP算法的详细介绍，这里不再赘述，网上很多，这里直接列出模板。

class Solution {
    public int strStr(String haystack, String needle) {
        if(needle.equals("")) return 0;
        //KMP算法，前缀表
        int j=0;
        int[] next=new int[needle.length()];
        getNext(next,needle);
        for(int i=0;i<haystack.length();i++){
            while(j>0&&haystack.charAt(i)!=needle.charAt(j)){
                j=next[j-1];
            }
            if(haystack.charAt(i)==needle.charAt(j)){
                j++;
            }
            if(j==needle.length()){
                return i-needle.length()+1;
            }
        }
        return -1;
    }

    public void getNext(int[] next,String needle){
        int j=0;
        next[0]=j;
        for(int i=1;i<needle.length();i++){
            //处理前缀不相同的情况
            while(j>0&&needle.charAt(i)!=needle.charAt(j)){
                j=next[j-1];
            }
            if(needle.charAt(i)==needle.charAt(j)){
                j++;
            }
            next[i]=j;
        }
    }
}

459. 重复的子字符串

这题是简单题我是没想到的。

==枚举==

如果一个长度为n的字符串s可以由它的一个长度为n`的子串重复多次构成，那

  • n是n`的倍数
  • s`是s的前缀
  • 对于任意$i\in [n,n]，有s[i-n]=s[i]$，

由于子串至少会重复一次，故只需枚举[1,n/2]范围内即可。

class Solution {
    public boolean repeatedSubstringPattern(String s) {
        int len=s.length();
        //子串至少会重复一次，只需枚举【1，n/2】
        for(int i=1;i*2<=len;i++){
            if(len%i==0){
                boolean flag=true;
                for(int j=i;j<len;j++){
                    if(s.charAt(j)!=s.charAt(j-i)){
                        flag=false;
                        break;
                    }
                }
                if(flag){
                    return true;
                }
            }
        }
        return false;
    }
}

==KMP+公式法==

KMP关键就在于求next数组，前面已经讲过，这里不再赘述，关于重复子字符串，有一个公式

len % (len - (next[len - 1] + 1)) == 0

数组长度减去最长相同前后缀的长度相当于是第一个周期的长度，也就是一个周期的长度，如果这个周期可以被整除，就说明整个数组就是这个周期的循环。

具体证明可搜索。

class Solution {
    public boolean repeatedSubstringPattern(String s) {
        //公式：
        //len % (len - (next[len - 1] + 1)) == 0 
        //数组长度减去最长相同前后缀的长度相当于是第一个周期的长度，也就是一个周期的长度，如果这个周期可以被整除，就说明整个数组就是这个周期的循环。
        int len=s.length();
        int[] next=new int[len];
        getNext(next,s);
        if(next[len-1]!=0&&len%(len-next[len-1])==0){
            return true;
        }
        return false;
    }

    public void getNext(int[] next,String s){
        int j=0;
        next[0]=j;
        for(int i=1;i<next.length;i++){
            while(j>0&&s.charAt(i)!=s.charAt(j)){
                j=next[j-1];
            }
            if(s.charAt(i)==s.charAt(j)){
                j++;
            }
            next[i]=j;
        }
    }
}
